VR实训项目Demo

物体的抓取与事件交互

提交于 2017/05/24

在VR项目中，与虚拟物体的交互是一个十分重要的课题。为了之后在项目中能够进行更自然的物体交互，现试做一个Demo用于测试现有的物体交互功能。

这个Demo中使用的技术包括：

  • Interactable Object
  • Controller Interacts

Demo场景展示：

Demo场景 场景的Hierarchy结构

场景中的功能说明：

使用右手手柄触碰物体后按下抓握按钮实现物体的抓取。
在抓握状态下或者未抓取状态下触摸物体并按下扳机键，将触发物体的事件，在控制台输出信息。
在按下抓握键的情况下去触碰物体将可以推动物体。

部分代码展示： 


using UnityEngine;
using VRTK;

public class Demo_Sphere : VRTK_InteractableObject {

    public override void StartUsing(GameObject currentUsingObject) {
        base.StartUsing(currentUsingObject);
        Debug.Log("Using Object!");
    }

    public override void StopUsing(GameObject currentUsingObject) {
        base.StopUsing(currentUsingObject);
        Debug.Log("Stop Using Object!");
    }
}
